# zaprett-repo-utils

Небольшой набор инструментов, упрощающий работу с [репозиторием](https://github.com/CherretGit/zaprett-repo).

## Доступные команды

| Команда  | Описание                                             | Использование                  |
|----------|------------------------------------------------------|--------------------------------|
| `check`  | Проверка корректности манифестов (JSON, хешей и файлов)      | `zaprett-repo-utils check`     |
| `update` | Обновление / исправление хешей                       | `zaprett-repo-utils update`    |
| `new`    | Добавление нового файла                              | `zaprett-repo-utils new <path>`|

## Дополнительно

Каждая команда поддерживает собственные аргументы и опции.

Для получения подробной информации используйте:

```bash
zaprett-repo-utils <команда> --help
